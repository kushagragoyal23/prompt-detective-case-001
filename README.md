# 🕵️ Prompt Detective: CASE-001 – The Lost AI

> An interactive AI learning mini app that teaches Prompt Engineering through detective-style gameplay.

**Developed by GOYAL DIGITAL LABS™**

---

## 📖 Overview

Prompt Detective: CASE-001 – The Lost AI is an interactive AI learning experience designed to make Prompt Engineering engaging through gameplay.

Instead of reading traditional tutorials, players solve an AI mystery by progressing through four investigation phases. Each phase introduces practical Prompt Engineering concepts while keeping the experience immersive and story-driven.

---

## 🎯 Objective

- Solve the mystery of the Lost AI.
- Learn real Prompt Engineering concepts.
- Think critically while interacting with AI.
- Experience learning through gameplay.
